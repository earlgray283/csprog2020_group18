# csprog2020 group18

## Requirements
+ make
```console
$ sudo apt install build-essential
$ make --version
```

+ javac, java
```console
$ sudo apt install default-jdk
$ javac --version
$ java --version
```

+ openjfx
```console
$ sudo apt install openjfx
$ dpkg -L openjfx
```

## Commands
基本的にコンパイルや実行などは `make` で行ってください。そっちのほうが楽ですし、メンバー全員で統一することができます。

### Run
```console
$ make run
```
